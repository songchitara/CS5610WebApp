Overview Top

In this project, I developed a web application which can help users to search for movies, look up top rated movies, movies now playing, and help user to remember those movies he/she likes. Also, user can like other users and view their favorite movies.

The technology I used is MEAN stack, hosted on the OpenShift cloud, and the movie data fetched through the TMDb API.

How to Use

Home Top

Anyone can use the Home page without login. Users can see movies organised in Top Rated or Now Playing groups. The Top Rated movies are movies top rated calculated by the TMDb website. The Now Playing movies are movies now playing in the cinema.

Click on the Top Rated or Now Playing button to see these two different groups of movies. The movies are displayed in grids view with their thumbnail posters and scores. The Add button is for adding the movie to user's favorite, as you will imagine, only loggedin user can use this function.

Search Top

Similar to home page, users don't have to log in to user the search funciton. To search for a particular movie, type full or partial movie name in the search bar, click Search button, then all the related results will display below. The results are shown will thumbnail posters, movie titles, ratings and year of release. There is a Filter input area next to the search bar, use this to filter the results by keywords.

In each result row, at the rightmost there is a blue star button, a logged in user can use this button to add the movie to their favorite movies. However, guests can not use this function.

Register Top

Register a user is easy. Click the register link on the top right of navigation bar, enter the username and passwords(twice) to register a new user. The page will notify you if your username is too short or too long, or your two passwords don't match. If you register for a username which has already existed, you will not be able to successfully register.

Profile Top

The Profile Page can only be used by user who has logged in. In the page, there are mainly three sections.
1) Section for displaying the user's favorite movies.
2) Section for displaying the user's liked users.
3) Section for displaying the other user's liked movies.

The two blue toggle button at the top of the first two sections are to toggle the displays. And the number in paranthesis shows the count of favorite movies/users.

Below the Toggle Favorite Movies button is the first section, which shows the current logged in user's favorite movies. In the displaying table, you will see the name, year, thumbnail of the movies, and when you click the Movie Details button you will be navigated to the Movie Details page where you can see more details of the movie. The red Delete will remove the particular movie from the favorite movie list.

Below the Toggle People I Liked button, an area will be displaying all the users the current logged in user has liked. The current user can use the "red delete" buttons to remove any users from his/her liked list. You'll also see a "green eye" button in each row of user, when click on it, you will see an area down below with title "Other User's Movies" which displays the selected user's favorite movies.

In the "Other User's Movies" section, you'll see a grid view of movies with thumbnails and title, user can click on any of the thumbnails or titles to jump to the Details Page to see the details of that particular movie.

Details Top

In the Details page, the user can view the details of the selected movie. The page will display the movie's poster at the top-left of the page, and the movie details information like the pilot and those people who also liked this movie.

Some of the information like the title of the movie and the users who also liked the movie will be directly fetched from the database. And the information like poster, the pilots will be fetched from the web through the TMDb API.

Down below the "People Who Also Liked This Movie" title, you will see a table that shows the all other users who have also like this movie. At each row, there are two buttons to the right. The blue Add to Friends button can let you to add the particular user into your friend list. The yellow Favorite Movies, when you click on it, the section below named Favorite Movies will display all the favorite movies of the particular user.

Database Top

The MongoDB database is used for this project. Mainly I created two collections/tables to store the data, one is the UserModel and the other is MovieModel. Let first see the

The User table is defined like below:
Field name Type Description
username String The username of a user
password String The password of the user
friendList [String] A list of users that the current logged in user has liked.
favoriteMovieList [Number] A list of movie ids that the current logged in user has liked.

The Movie table is defined as below:
Field Name Type Description
id Number The id of movie
title String The title/name of movie
year String The release year of movie
rating Number The rating of movie
posterURL String The url where to get the poster of movie
userList [String] A list of users who have liked the movie
