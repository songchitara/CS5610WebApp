AngularJS Routing - Basic Route

Description: This experiment I will implement the basic route between two pages: home.html and favorite.html

Instruction: Click Home or Favorite to route to different pages.

Home | Favorite

Implementation

In the html, we first create the Home and Favorite navigation links. 

<nav>
<a href="#/home">Home</a> |
<a href="#/favorite">Favorite</a>
</nav>

THen we create the ng-view placeholder. 

<div ng-view>
</div>

In the javascript, now we set the ngRoute parameter.

 var app = angular.module('MovieApp',['ngRoute'] );

Then we need to config the route by set the app.config. We use the $routeProvider service, when '/home', navigate to the home.html; when '/favorite', navigate to favorite.html, otherwise all navigate to home.html. 

app.config(['$routeProvider',
    function ($routeProvider) {
        $routeProvider.
        when('/home', {
            templateUrl: 'partials/home.html'
        }).
        when('/favorite', {
            templateUrl: 'partials/favorite.html'
        }).
        otherwise({
            redirectTo: '/home'
        })
    }
])
