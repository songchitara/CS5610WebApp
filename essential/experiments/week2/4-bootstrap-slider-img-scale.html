Descriptions: In last exmperiment, I used the Bootstrap Carousel to implement a slider. However, the image in the slide got wierd stretched. This experiment I will solve the problem.

The size of the image shown in the slide can be set by the .carousel-inner > .item > img CSS selector. We first comment out the two default setting: position: absolute and min-width: 100%, then we set the margin to auto. See code snippet below: 

    .carousel-inner > .item > img {
//position: absolute;
  top: 0;
  left: 0;
  height: 100%;
//min-width: 100%;
margin: auto;
      
}
