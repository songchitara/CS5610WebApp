jQuery Hide and Show

Descriptions: In the previous experiment, we've implemented sorting paragraphs by dragging them. However, if there are many paragraphs (or some of paragraphs are long), it will be pretty inconvenient to drag one paragraph and place it to another location, since you have to drag and scroll the page and the same time. To solve this, I'm using the jQuery slide show and hide features, so that we can first hide the content of the paragraphs, then just drag the titles, and after reordering done, show the content of the paragraphs.

Instruction: Click on the each title of the paragraph to toggle hide/show the content of paragraph. For convenience, we can first click the button to hide all the paragraphs (only remain the titles), then we can drag around the titles to the desired order. Note that at any time, we can click on any of the title to show/hide the paragraph, and we can either drag a title with paragraph hided or a completed showed paragraph. Once the paragraphs are in our desired order, click on the button of show all the paragraphs again.

Implementation:

We first mark the paragraph titles' ids as "flip1", "flip2", .... And we mark the related paragraphs' ids as "p1", "p2", ... 

<section id="sec1" draggable="true"> 
            <h3 id="flip1">Section One</h3>
            <p id="p1" class="para">
            Lorem ipsum dolor sit amet, consectetur adipiscing ....
        </p>
</section>

Then we call the slideToggle jQuery functions on the paragraphs when the according title is been clicking on. 

$(document).ready(function(){
    $("#flip1").click(function () {
        $("#p1").slideToggle();
    });
    $("#flip2").click(function () {
        $("#p2").slideToggle();
    });
    $("#flip3").click(function () {
        $("#p3").slideToggle();
    });
    $("#flip4").click(function () {
        $("#p4").slideToggle();
    });
    $("#flip5").click(function () {
        $("#p5").slideToggle();
    });
    $("#flip6").click(function () {
        $("#p6").slideToggle();
    });
});

For the and buttons, first create these two buttons in HTML: 

<input type="button" id="bttn_hide" value="Hide All" />
<input type="button" id="bttn_show" value="Show All" />

We then call the slideUp() jQuery function on the button: 

$(document).ready(function () {
    $("#bttn_hide").click(function () {
        $(".para").slideUp();
    });
});

And we call the slideDown() jQuery function on the button: 

$(document).ready(function () {
    $("#bttn_show").click(function () {
        $(".para").slideDown();
    });
});
