Bootstrap Slider

Descriptions: In this exmperiment, I will use the Bootstrap Carousel to implement a slider.

Instructions: The slider can auto slide every 3 secs. We can also manually slide by clicking the left arrow or right arrow to slide left or right. Also, you can click on the small circles to jump to the specific slide. Hover over the pointer to the image to stop the auto sliding.

In order to use Bootstrap, we first include the links in the <head>: 

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/css/bootstrap.min.css">

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/css/bootstrap-theme.min.css">

Then to use the Bootstrap Javascript and jQuery, we are going to include the following two link (we put the at the end of the file to make the page load faster): 

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/js/bootstrap.min.js"></script>

Now we can start using the Bootstrap Carousel. First we create a <div> with class name "carousel slide". Set data-ride attribute to "carousel" to make the slide auto cycling. The data-interval attribute is to set the time delay between auto cycling items (in "ms"). 

<div id="carousel-example-generic" class="carousel slide" data-ride="carousel" data-interval="2000">
    ...    
</div>

In the <div> we've just created, first add the indicator feature, which you can click on the small circles to go to the specific slide. 

<ol class="carousel-indicators">
<li data-target="#carousel-example-generic" data-slide-to="0" class="active"></li>
<li data-target="#carousel-example-generic" data-slide-to="1"></li>
<li data-target="#carousel-example-generic" data-slide-to="2"></li>
<li data-target="#carousel-example-generic" data-slide-to="3"></li>
</ol>

THe code below is to wrapping the slides: 

                <div class="carousel-inner ">
                <div class="item active">
                <img src="../../images/img_week2/mandala-thousand-arms.jpg" alt="...">
                <div class="carousel-caption">
                <h3>Mandala of Thousand Arms Avalokitesvara</h3>
                    </div>
                </div>
                    .
                    .
                    .
                </div>

Finally, we will add the left and right control arrows. 

                <a class="left carousel-control" href="#carousel-example-generic" role="button" data-slide="prev">
                <span class="glyphicon glyphicon-chevron-left"></span>
            </a>
                <a class="right carousel-control" href="#carousel-example-generic" role="button" data-slide="next">
                <span class="glyphicon glyphicon-chevron-right"></span>
            </a>
