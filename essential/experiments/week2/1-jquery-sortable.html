jQuery Sortable

Descriptions: Last week, I implemented sorting paragraphs/objects by using HTML5 drag and drop features, however it needs to create a temporary area to hold the temp paragraph. This time, I will take advantages of the jQuery library to make paragraphs directly sortable without any temp areas.

Instruction: Simply drag any paragraphs to re-order.

Implementation:

Since I'm using the existed jQuery sorting API from web, what we need to do is first include the two javascript sources in the <head>.

Below is the code snippet:

<script src="//code.jquery.com/jquery-1.10.2.js"></script>
<script src="//code.jquery.com/ui/1.11.2/jquery-ui.js"></script>

Then mark the div that contains the paragraphs as <id="sortable">. Below is the code snippet: 

<div id="sortable">
<section id="sec1" draggable="true">
<h3>Section One</h3>
<p id="p1">
            Lorem ipsum dolor sit amet, consectetur adipiscing elit....
    </p>
</section>
</div>

Then in the <head> we call the two funcions sortable and disableSelection() on the marked div.

<script>
$(function () {
    $("#sortable").sortable();
    $("#sortable").disableSelection();
});
</script>
