Project Related: Relationship Visualization with Images

Descriptions:

In this experiement, I will use another type of graph in CytoscapeJS which can specify our own image as the graph nodes. This is very helpful to my project that render the movies and user relationships with the movie posters as the graph node.

Instructions:

Click HERE to the experiment page. You'll see the visualized graph with our own images be the nodes. Again, use mouse wheel to zoom in/out the graph and each node can be dragged.


Implementation

The images we want to put as nodes can be specified with the cytoscape() method. With the method, use the CSS style to specify the background-image link. 

var cy = cytoscape({
        container: document.getElementById('cy'),
        style: cytoscape.stylesheet()
            .
            .
            .
          .selector('#bird')
            .css({
                'background-image': 'https://farm8.staticflickr.com/7272/7633179468_3e19e45a0c_b.jpg'
            })
          .selector('#cat')
            .css({
                'background-image': 'https://farm2.staticflickr.com/1261/1413379559_412a540d29_b.jpg'
            })
            .
            .

The nodes and edges specifications are the same as in previous experiment. 

elements: {
    nodes: [
        { data: { id: 'cat' } },
        { data: { id: 'bird' } },
        .
        .
        .
    ],
    edges: [
        { data: { source: 'cat', target: 'bird' } },
        { data: { source: 'bird', target: 'ladybug' } },
        .
        .
    ]
}
