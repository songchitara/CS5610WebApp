Project Related: Calendar and Events

Descriptions:

Calendar is quite useful for displaying events. In this experiment, I will use the FullCalendar to create a calendar and customize the events. Later in my project, I could add the event/movie show into the Calendar and displaying in the home page.

Instructions:

Click HERE to the experiment page. You'll see a calendar with events on it.


Implementation

In the HTML, first of all, we need to include four libraries. Note that the jQuery and Moment must be loaded before FullCalendar's JavaScript. 

<link rel='stylesheet' href='fullcalendar/fullcalendar.css' />
<script src='lib/jquery.min.js'></script>
<script src='lib/moment.min.js'></script>
<script src='fullcalendar/fullcalendar.js'></script>

Then in the body, place the calendar placeholder. 

    <div id='calendar'></div>

Then in the JavaScript file, call $('#calendar').fullCalendar() function after the page has initialized. 

$(document).ready(function() {
    $('#calendar').fullCalendar({
        ...
    })
});

Withn the fullCalendar() function, add the JSON format event to display the events on the calendar. 

    $('#calendar').fullCalendar({
        events: [
        {
            title: 'event1',
            start: '2015-04-07'
        },
        {
            title: 'event2',
            start: '2015-04-15',
        },
        {
            title: 'event3',
            start: '2015-04-19T12:30:00',
            allDay: false // will make the time show
        }
        ]
    })
