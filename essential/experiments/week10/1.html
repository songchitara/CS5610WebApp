Project Related: Relationship Visualization

Descriptions:

It is always a good idea to visualize things. In this experiment, I will use the CytoscapeJS to create a nice visualized JSON relationships. Later I will use this in my project to visualize the relations and connections among users.

Instructions:

Click HERE to the experiment page. You'll see the visualized graph created from JSON data. Scroll mouse wheel to zoom in/out the graph. Also, each node in the graph is draggable.


Implementation

First in the HTML page, include the CytoscapeJS library. 

<script src="http://cytoscape.github.io/cytoscape.js/api/cytoscape.js-latest/cytoscape.min.js"></script>

Then in the Javascript file, put the JSON configurations. The nodes is a array of nodes in the graph, and edges represent the edges of the graph.The source and target in the edges together can represent the direction of the edges, while strength specify the percentage of color strength. 

        elements: {
            nodes: [
              { data: { id: 'j', name: 'Alice', weight: 65, faveColor: '#6FB1FC', faveShape: 'triangle' } },
              { data: { id: 'e', name: 'Bob', weight: 45, faveColor: '#EDA1ED', faveShape: 'ellipse' } },
              { data: { id: 'k', name: 'Chris', weight: 75, faveColor: '#86B342', faveShape: 'octagon' } },
              { data: { id: 'g', name: 'Dan', weight: 70, faveColor: '#F5A45D', faveShape: 'rectangle' } }
            ],
            edges: [
              { data: { source: 'j', target: 'e', faveColor: '#6FB1FC', strength: 90 } },
              { data: { source: 'j', target: 'k', faveColor: '#6FB1FC', strength: 70 } },
              { data: { source: 'j', target: 'g', faveColor: '#6FB1FC', strength: 80 } },
              { data: { source: 'e', target: 'j', faveColor: '#EDA1ED', strength: 95 } },
              { data: { source: 'e', target: 'k', faveColor: '#EDA1ED', strength: 60 }, classes: 'questionable' },
              { data: { source: 'k', target: 'j', faveColor: '#86B342', strength: 100 } },
              { data: { source: 'k', target: 'e', faveColor: '#86B342', strength: 100 } },
              { data: { source: 'k', target: 'g', faveColor: '#86B342', strength: 100 } },
              { data: { source: 'g', target: 'j', faveColor: '#F5A45D', strength: 90 } }
            ]
        }
