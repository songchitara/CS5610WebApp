Project Related: Google Map API

Descriptions:

In this experiement, I will use the google map api to mark favorite place. Later in the project, a favorite theater could be shared in through map among users.

Instructions:

As below you see the google map with a marker, also an infoWindow that gives the title of the favorite place.


Implementation

First in the HTML header tag, load the google api. Then in the body, create the container for map. 

<script src="http://maps.googleapis.com/maps/api/js?key=AIzaSyDThQNXNA1K4xjoVrdMzUVSv-GCBb8CLRk&sensor=false"></script>

<div class="container" id="googleMap"></div>

In the JavaScript file, use google.maps.LatLng() to create the center of our location. 

var myCenter = new google.maps.LatLng(42.360379, -71.064756);

In the initialize() function, first specify the properties of the map. 

function initialize() {
    var mapProp = {
        center: myCenter,
        zoom: 15,
        mapTypeId: google.maps.MapTypeId.ROADMAP
    };

Then use google.maps.Map() to create a map instance. 

                var map = new google.maps.Map(document.getElementById("googleMap"), mapProp);

User google.maps.Marker() to create marker on the map and user setMap() to save the marker on the map. 

var marker = new google.maps.Marker({
        position: myCenter,
        animation: google.maps.Animation.BOUNCE
    });

 marker.setMap(map);

Finally use google.maps.InfoWindow to create info window. The content field is the message of the infowindow. 

    var infowindow = new google.maps.InfoWindow({
        content: "Favorite Place 1"
    });

    infowindow.open(map, marker);

Finally call the initialize() function. 

google.maps.event.addDomListener(window, 'load', initialize);
