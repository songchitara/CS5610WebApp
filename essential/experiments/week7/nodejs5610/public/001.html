NodeJS GET - all

Descriptions:

In this page, I experimented nodejs GET requests to get all the users informations.

Instructions:

After the page loaded, all the name and role information of users will be shown in the table.


Name Role
{{user.name}} {{user.role}}



Implementations

To get all users' information, first in html page add the table placeholder. 

<tbody>
<tr ng-repeat="user in users">
<td>{{user.name}}</td>
<td>{{user.role}}</td>
.
.
.
</tr>
</tbody>

Then at the client side, in clientApp.js, use the code below to send GET request to the server by using $http.get() 

$http.get('/api/user')
.success(function (response) {
    $scope.users = response;
})

At the server side, in server.js file, add the code below to receive the request and return the users json as the response. 

app.get('/api/user', function (req, res) {
    res.json(users);
})
