jQuery Responsive Tables

Descriptions: In this exmperiment, I will use jQuery mobile to implement responsive tables.

Instructions: While resizing the page, the table will hide columns from the right to left if reducing the width, and will resume from left to right if increase the width. Also, click the to customize the columns to show.

Implementation:

First include the jquery mobile library. 

<script src="http://code.jquery.com/mobile/1.4.5/jquery.mobile-1.4.5.min.js"></script>

For the <table> element, there are several attributes to set. First is data-role attribute, we set it to "table". Second we set class to "ui-responsive" to declare our table as responsive. Then we set the data-mode to "columntoggle". 

<table data-role="table" data-mode="columntoggle" class="ui-responsive" id="myTable">

We can also set the data-priority attribute of <th> to assign the priority of the column (meaning the priority not be hidden). 

<th data-priority="1">CustomerID</th>
<th>CustomerName</th>
<th data-priority="2">Address</th>
<th data-priority="3">City</th>
<th data-priority="4">PostalCode</th>
<th data-priority="5">Country</th>
<th data-priority="6">ContactName</th>
