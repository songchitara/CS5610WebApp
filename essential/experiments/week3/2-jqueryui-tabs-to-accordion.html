jQuery UI Responsive Tabs

Descriptions: jQuery UI tabs are very good to use, however, I found that once the screen width reduces, the tabs will not remain their original posistions. So in this experiment, I will implement responsive design between tabs and accordion to solve the problem.

Instructions: When the screen is big (desktop/laptop display), the content will be shown in tabs. If the screen is small (mobile device), the content will be shown in accordion style. Also, both the tabs and accordion can be sortable.

Implementation:

In the html, create two <div>: <div id="accordion"> and <div id="tabs">.

The jquery code for creating the accordion behavior is as below: 

$(function () {
    $("#accordion")
      .accordion({
          header: "> div > h3",
          collapsible: true
      })
      .sortable({
          axis: "y",
          handle: "h3",
          stop: function (event, ui) {
              // IE doesn't register the blur when sorting
              // so trigger focusout handlers to remove .ui-state-focus
              ui.item.children("h3").triggerHandler("focusout");
              // Refresh accordion to handle new order
              $(this).accordion("refresh");
          }
      });
});

To implement jQueryUI tabs, using the code below: 

                $(function () {
    var tabs = $("#tabs").tabs();
    tabs.find(".ui-tabs-nav").sortable({
        axis: "x",
        stop: function () {
            tabs.tabs("refresh");
        }
    });
});

Then I'm going to making the page responsive by checking the $(window) size. First, I will tell the browser when loading the page, at which widths should display tabs or accordions. 

$(function () {
    var accordionDom = $("div#accordion");
    var tabsDom = $("div#tabs");
    if ($(window).width() <= 700) {
        accordionDom.attr("style", "display:block");
        tabsDom.attr("style", "display: none");
    }
    else {
        $(function () {
            accordionDom.attr("style", "display:none");
            tabsDom.attr("style", "display: block");
        });
    }
});

Then I'm using $(window).resize() to make the page responsive to the width changes. 

$(window).resize(function () {
    var accordionDom = $("div#accordion");
    var tabsDom = $("div#tabs");
    if ($(window).width() <= 700) {
        $(function () {
            accordionDom.attr("style", "display:block");
            tabsDom.attr("style", "display: none");
        });
    }
    else {
        $(function () {
            accordionDom.attr("style", "display:none");
            tabsDom.attr("style", "display: block");
        });
    }
});
