Responsive Design

In this page, I will experiment on the responsive design.

Description:

This page is responsible for different screen sizes: computer, tablet and phone. Drag to resize the page to see how this page is responding to different screen sizes.

Laptop/Desktop Display - The biggest screen, everything can be shown.
Tablet Display - The medium screen, everything can still be shown, in relative smaller size.
Phone Display - The smallest screen, the left-side navigation area will hide, and a small dropdown navigation area will be created.

Implementation:

To respond to the screen size, we first use @media to create the different breakpoints for different screen size.

The following code snippet is for creating the display breakpoints for laptop/desktop, tablet and phone respectively. 


    @media screen and (min-width: 1000px) {
        .......
    }

    @media screen and (min-width: 700px) and (max-width: 1000px) {
        .......
    }

    @media screen and (max-width: 700px) {
        .......
    }

To be responsive from the laptop/desktop to the tablet, we only need to set the relevant container and row/column to a relative smaller size.

Below is the size of container and row/col for laptop/desktop screen:


@media screen and (min-width: 1000px) {
    .container {
        width: 840px;
        margin: auto;
    }
    .row {
        width:840px;
        clear:both;
    }
    .col-3 {
        width:210px;
        float:left;
    }
    .col-9 {
        width:630px;
        float:left;
    }
}

In order to fit in the tablet screen, we use the relative smaller width as below for tablet:


@media screen and (min-width: 700px) and (max-width: 1000px) {
    .container {
        width: 600px;
        margin: auto;
    }
    .row {
        width:600px;
        clear:both;
    }
    .col-3 {
        width:150px;
        float:left;
    }
    .col-9 {
        width:450px;
        float:left;
    }
}

Moving from tablet from phone is more tricky. We first need to hide the normal navigation area when detecting the phone display:


@media screen and (max-width: 700px) {
    body nav {
    display: none;
    }
}

Then in the html page, we need to create a new navigation area, the code is as below. 


    <div id="phone_nav">
        <ul>
            <li>
                <a href="">Navigation</a>
                <ul>
                    <li><a href="#sec1">Section 1</a></li>
                    <li><a href="#sec2">Section 2</a></li>
                    <li><a href="#sec3">Section 3</a></li>
                </ul>
            </li>
        </ul>
    </div>

Then we need to css the new navigation area to make it as a dropdown display.

In the css file, we first zero the default padding and margin. 


    ul {
        padding: 0;
        margin: 0;
    }

Then we want to make each list item lay out horizontally. Also, we want to set the origin position is the parent position for all sub-lists. Below is the css code: 


    li {
        display: inline;
        position: relative;
    }

Now for the lists within the list, we want to position them absolutely, and place them over everything, and hide them. 


    ul ul {
        position: absolute;
        display: none;
    }

Finally, in order to reveal the sublists when hover over the mouse, we using the following css code: 


    li:hover ul {
        display: block;
    }
