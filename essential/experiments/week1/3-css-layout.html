Implementation:

I'm using the "float:left" attribute to create multiple columns. To make two column, one to fit the navigation area, another to fit the article area, simply add the float:left attribute to both nav and article selectors in the CSS file.

Below is the code snippet in the CSS file:

nav {
    float: left;
    background-color:#93cac0;
    height:1000px;
    width:150px;
    padding:10px;
}
                
article {
    width:650px;
    float:left;
    padding:10px;
}
